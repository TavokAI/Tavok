02 — Principle
Accent Restraint
Gold and cyan are strong colors. If they're on every icon, border, label, and avatar — nothing feels special. Gold = selected/primary action only. Cyan = streaming/live only. Everything else stays grayscale. When streaming kicks in, the cyan should feel like something just woke up.
Before — color everywhere

03 — Principle
Scanlines: Felt, Not Seen
The CRT effect should be subliminal. If someone can see the lines when reading text, they're too strong. Kill the RGB color fringe entirely — it just muddies the type.
Before — heavy scanlines + RGB fringe

The fix
Replace: rgba(0,0,0,0.25) → rgba(0,0,0,0.04)

Keep: background-size: 100% 2px (the line spacing is fine)

05 — Principle
Borders vs. Gaps
Too many #222 borders create a cage-bar effect. Replace most with 1–2px gaps that expose the darker background underneath. The darkness IS the separator.
